# Easter Eggs Design — Amber Code

Дата: 2026-04-18

## Концепция

Три пасхалки на сайте, все в эстетике ретро-пиксель + янтарь.

### 1) Загадка Эйлера → Анекдот (центральная)

**Суть.** На HomePage в разных местах спрятаны **7 маленьких «мостов»** — крошечные кликабельные точки (8×8px, opacity ~0.15, hover — янтарная подсветка). Пользователь должен найти и кликнуть все 7.

**Награда.** При 7/7 открывается `AnecdoteModal`:
- заголовок ретро-шрифтом: «Леонард Эйлер передаёт тебе анекдот»
- мини SVG с графом Кёнигсберга (4 вершины, 7 рёбер)
- текст анекдота про 100 розовых шариков для пинг-понга
- кнопка «Так для чего они были нужны?» → раскрывает: *«Этого никто не узнает. Как и рассказ про вагон метро или последний день в спортивном лагере…»*
- подпись мелко: *«Курица из Мафии оценила бы эту пасхалку»*
- фоновые **100 розовых шариков для пинг-понга** (canvas-конфетти с гравитацией)

**Meta-joke:** задача Эйлера не имеет решения, анекдот не имеет развязки. Сложенные вместе — двойная шутка «ответа не будет».

### 2) Konami-код → Янтарный дождь

Слушатель на window: `↑ ↑ ↓ ↓ ← → ← → B A`. При совпадении запускается canvas fixed fullscreen, pointer-events-none, с падающими янтарными кристаллами ~10 сек.

### 3) Консольный намёк

При загрузке сайта в `console.log` печатается ASCII-янтарь + строка *«7 bridges. 0 solutions. 100 ping-pong balls. amber://find»*. Намёк для тех, кто открывает DevTools.

## Компоненты

```
├── AnecdoteModal.tsx          # оверлей + карточка с анекдотом
├── PingPongConfetti.tsx       # 100 розовых шариков canvas
└── AmberRain.tsx              # янтарный дождь canvas

src/lib/
├── useAchievements.ts         # localStorage state пасхалок
├── consoleArt.ts              # ASCII-арт при старте
└── useKonamiCode.ts           # хук для Konami-последовательности
```

## localStorage keys

- `amber:bridges-found` — `string[]`, ID найденных мостов
- `amber:anecdote-unlocked` — `boolean`
- `amber:konami-triggered` — `boolean` (лайт-счётчик, не обязателен)

## Точки крепления

- `App.tsx` — добавить `<EasterEggsContainer />` внутри `<ToastProvider>`
- `HomePage.tsx` — добавить `<EulerBridges />` в конец (он сам разместит точки в fixed-координатах относительно вьюпорта или в углах блоков через `position: absolute`)
- `Header.tsx` — если `amber:anecdote-unlocked`, рядом с «Amber Code» крошечный розовый шарик (🎈) с `title`-tooltip

## UX-детали

- Все анимации уважают `prefers-reduced-motion` (уже настроено глобально в index.css)
- Модалка: `Escape` закрывает, клик по оверлею закрывает, фокус — на кнопку закрытия
- Мосты пропадают (становятся `unfound: false`) после разгадки
- Если все 7 уже найдены и модалка закрыта — при повторной «активации» (например, добавим команду в консоль `amber.tell()`) модалку можно открыть снова
